# Online ordering system: a design-first approach
> Minimal API for online ordering made with code generation

## Tech stack
- Programming language: [Go](https://go.dev/)
- Web framework: [ogen](https://ogen.dev/)
- Database: [MongoDB Go Driver](https://www.mongodb.com/docs/drivers/go/current/)
- Documentation: [OpenAPI Specification 3.0](https://swagger.io/specification/)
- Configuration: [Viper](https://github.com/spf13/viper)
- Logging: [Zap](https://github.com/uber-go/zap)
